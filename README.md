# 📝 To-Do App

## 📌 Project Description
This is a basic To-Do App built using HTML, CSS, and JavaScript. It allows users to add, edit, delete, and organize tasks with a specific date and time, and mark them as completed.

## 🚀 Features
- Add new tasks
- Set date and time for tasks
- Organize tasks into categories
- Edit existing tasks
- Mark tasks as completed
- Delete tasks

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download or clone the repository.
2. Open the `index.html` file in your browser.
3. Start adding and managing your tasks.

## 📚 Concepts Used
- DOM Manipulation
- Event Handling
- Conditional Statements
- Functions
